## 运行测试命令

```bash
# 运行所有测试
cd backend
mvn test

# 运行特定测试类
mvn test -Dtest=JwtUtilTest
mvn test -Dtest=AuthControllerTest

# 运行测试并生成报告
mvn test surefire-report:report
```
